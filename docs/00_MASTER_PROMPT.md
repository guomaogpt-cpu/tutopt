# MASTER PROMPT — Tutopt

## Project

Tutopt — это современная веб-платформа для размещения оптовых объявлений в Кыргызстане.

Основная задача проекта — соединить производителей, импортеров, дистрибьюторов, оптовые компании и покупателей.

Это НЕ интернет-магазин.

Это НЕ маркетплейс со встроенной оплатой.

Это НЕ CRM.

Это НЕ ERP.

Это НЕ система логистики.

Это НЕ тендерная площадка.

Это НЕ система импорта из Китая.

Платформа похожа на Lalafo и Bazar, но ориентирована исключительно на B2B-оптовую торговлю.

---

# Основные принципы

Cursor является разработчиком проекта.

Cursor НЕ принимает архитектурных решений самостоятельно.

Перед выполнением любой задачи Cursor обязан:

1. Прочитать документацию из папки docs.

2. Следовать существующей архитектуре.

3. Не изменять структуру проекта без необходимости.

4. Не удалять существующий код без подтверждения.

5. Работать небольшими итерациями.

6. После завершения каждой задачи показывать:

   - изменённые файлы;

   - краткое описание изменений;

   - дальнейшие рекомендации.
