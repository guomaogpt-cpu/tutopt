# Tutopt — Roadmap

## 1. Принципы планирования

- **MVP first:** запуск с минимальным, но полным циклом (объявление → модерация → поиск → lead)
- Итерации по **2-4 недели** (спринты)
- Метрики из [01_PRODUCT_SPEC.md](./01_PRODUCT_SPEC.md) — критерии перехода между фазами

---

## 2. Фаза 0 — Foundation (2 недели)

**Цель:** готовая инфраструктура для разработки.

| Задача | Статус |
|--------|--------|
| Инициализация Next.js 15 + TypeScript | ⬜ |
| TailwindCSS + Shadcn UI | ⬜ |
| ESLint + Prettier | ⬜ |
| Prisma + PostgreSQL schema (базовая) | ⬜ |
| Docker Compose (app + db) | ⬜ |

| .env.example, README | ⬜ |
| Структура папок (app, components, lib, prisma) | ⬜ |

**Deliverable:** `docker compose up` → пустое приложение на localhost:3000

---

## 3. Фаза 1 — MVP Core (6-8 недель)

**Цель:** публичный каталог + регистрация + объявления + модерация.

### Спринт 1.1 — Auth & Users (2 нед.)

- Регистрация / вход (email, phone)
- Роли BUYER, SELLER
- Профиль покупателя
- Email verification
- Middleware защиты маршрутов

### Спринт 1.2 — Seller & Listings (2 нед.)

- Профиль компании продавца
- CRUD объявлений (wizard)
- Загрузка изображений (local/S3)
- Статусы: draft → pending → published
- Лимит 10 объявлений (free)

### Спринт 1.3 — Catalog & Search (2 нед.)

- Главная страница
- Каталог + категории (дерево)
- Страница объявления
- Профиль продавца
- Полнотекстовый поиск + базовые фильтры
- Пагинация, сортировка

### Спринт 1.4 — Leads & Admin (2 нед.)

- Заявки (leads): создание, просмотр
- Избранное
- Админ: модерация объявлений
- Админ: управление категориями
- Админ: пользователи (базово)
- Жалобы на объявления

**MVP Launch criteria:**

- [ ] 50+ тестовых объявлений
- [ ] Модерация работает end-to-end
- [ ] Поиск < 500 ms
- [ ] SSL на staging
- [ ] Базовый SEO (sitemap, meta)

## 4. Фаза 2 — Growth (8-10 недель)

**Цель:** удержание, монетизация, качество.

| Функция | Приоритет | Спринт |
|---------|-----------|--------|
| Верификация продавцов (документы) | P0 | 2.1 |
| SMS-верификация телефона | P0 | 2.1 |
| Расширенная аналитика продавца | P1 | 2.2 |
| PRO-подписка (Stripe / local payment) | P1 | 2.3 |
| Поднятие объявлений | P1 | 2.3 |
| Внутренний чат buyer ↔ seller | P1 | 2.4 |
| Redis кэш поиска | P1 | 2.2 |
| Баннеры на главной | P2 | 2.2 |
| SEO-лендинги (админка) | P1 | 2.3 |
| Отзывы о продавцах | P2 | 2.4 |
| OAuth (Google) | P2 | 2.1 |
| Уведомления email (leads, модерация) | P0 | 2.1 |
| Кыргызский язык (i18n) | P2 | 2.4 |

**KPI фазы 2:**

- 200+ активных продавцов
- 2 000+ объявлений
- 5 000 MAU
- Первый платящий PRO-продавец
