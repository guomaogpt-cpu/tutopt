# Tutopt — Карта страниц и маршрутов

## 1. Соглашения по URL

- Базовый домен: `https://tutopt.kg` (prod), `https://staging.tutopt.kg` (staging)
- Локаль: без префикса на MVP (русский по умолчанию); фаза 2: `/ky/...`
- Slug: латиница, kebab-case (`/category/stroitelnye-materialy`)
- ID объявлений в URL: slug + короткий id (`/listing/optovaya-prodazha-cementa-abc123`)

---

## 2. Публичные страницы

### 2.1 Главная `/`

| Блок | Описание |
|------|----------|
| Hero | Поисковая строка, CTA «Разместить объявление» |
| Популярные категории | Сетка иконок топ-12 категорий |
| Свежие объявления | Карусель / сетка последних 20 |
| Проверенные продавцы | Блок верифицированных компаний |
| SEO-текст | Короткий текст про оптовый маркетплейс КР |
| Footer | Ссылки, контакты, соцсети |

**SSR:** да (SEO)

---

### 2.2 Каталог `/catalog`

Список всех объявлений с фильтрами в сайдбаре (desktop) или drawer (mobile).

**Query params:** см. [07_SEARCH_FILTERS.md](./07_SEARCH_FILTERS.md)

---

### 2.3 Категория `/category/[slug]`

- H1: название категории
- Хлебные крошки
- Подкатегории (если есть дети)
- Список объявлений с фильтрами
- SEO-описание категории (из CMS)

**Примеры:** `/category/produkty-pitaniya`, `/category/odezhda-optom`

---

### 2.4 Объявление `/listing/[slug]`

| Блок | Описание |
|------|----------|
| Галерея | До 10 фото, zoom |
| Заголовок, цена, MOQ, единица | |
| Описание | Rich text (ограниченный) |
| Характеристики | Таблица key-value |
| Продавец | Карточка компании, ссылка на профиль |
| CTA | «Написать», «Позвонить», «В избранное» |
| Похожие объявления | Та же категория |
| Breadcrumb | Главная → Категория → Объявление |

**Schema.org:** Product, Offer

---

### 2.5 Профиль продавца `/seller/[slug]`

- Шапка: логотип, название, бейджи (verified, pro)
- Описание компании
- Контакты (после авторизации — полные)
- Вкладки: «Все объявления», «О компании»
- Рейтинг и отзывы (фаза 2)

---

### 2.6 Поиск `/search`

- Результаты по query `?q=...`
- Те же фильтры, что в каталоге
- Empty state с подсказками

## 8. Состояния UI (общие)

### 8.1 Empty states

- Нет объявлений в категории
- Пустое избранное
- Нет заявок
- Поиск без результатов

### 8.2 Error states

- 404 — страница не найдена
- 403 — нет доступа
- 500 — ошибка сервера
- Offline banner (PWA — фаза 3)

### 8.3 Loading

- Skeleton для карточек объявлений
- Spinner для форм

---

## 9. Навигация

### 9.1 Header (desktop)

- Логотип → `/`
- Каталог (mega-menu категорий)
- Поиск (autocomplete)
- «Разместить объявление» → `/seller/listings/new` или login
- Избранное, Вход / Аватар + dropdown

### 9.2 Mobile

- Bottom tab: Главная, Каталог, + (создать), Избранное, Профиль
- Hamburger: категории, статические страницы
