# Tutopt — Product Specification

## 1. Обзор продукта

**Tutopt** — B2B/B2C маркетплейс оптовой торговли для Кыргызстана. Платформа соединяет оптовых продавцов (поставщиков, дистрибьюторов, производителей) с покупателями (розничные магазины, HoReCa, перекупщики, мелкий опт).

Ориентир по UX и функциональности: **Lalafo**, **Bazar.kg**, с акцентом на оптовые условия, MOQ (минимальный заказ), прайс-листы и B2B-воронку.

### 1.1 Миссия

Упростить поиск оптовых поставщиков и закупку товаров в КР: прозрачные цены, фильтры по регионам, верифицированные продавцы, удобный каталог.

### 1.2 Целевая аудитория

| Сегмент | Описание | Потребность |
|---------|----------|-------------|
| Оптовый продавец | Склад, дистрибьютор, импортёр | Витрина, лиды, управление объявлениями |
| Розничный покупатель | Магазин, ларёк, маркетплейс-продавец | Оптовые цены, MOQ, доставка |
| HoReCa | Кафе, рестораны | Регулярные закупки, категории продуктов |
| Перекупщик | Мелкий опт, перепродажа | Быстрый поиск, сравнение цен |
| Администратор | Команда Tutopt | Модерация, аналитика, SEO |

### 1.3 География

- **MVP:** Бишкек, Чуйская область
- **Фаза 2:** Ош, Джалал-Абад, Каракол, вся КР
- **Языки:** русский (основной), кыргызский (фаза 2)

### 1.4 Валюта и единицы

- Основная валюта: **KGS (сом)**
- Дополнительно (фаза 3): USD для импортных позиций
- Единицы: штука, упаковка, короб, паллета, кг, литр

## 2. Ключевые возможности (MVP)

### 2.1 Каталог и объявления

- Создание объявлений с фото, описанием, ценой, MOQ, остатком
- Иерархия категорий (до 3 уровней)
- Статусы: черновик, на модерации, опубликовано, архив, отклонено
- Поднятие объявления (платная опция — фаза 2)

### 2.2 Поиск и фильтры

- Полнотекстовый поиск по названию и описанию
- Фильтры: категория, регион, цена, MOQ, наличие, верификация продавца
- Сортировка: по дате, цене, релевантности, популярности

### 2.3 Профили продавцов

- Страница компании: название, ИНН (опционально), адрес, контакты, рейтинг
- Верификация продавца (модерация документов)
- Список всех объявлений продавца

### 2.4 Коммуникация

- Кнопка «Написать продавцу» / «Позвонить» (телефон после авторизации)
- Внутренний чат (фаза 2)
- Заявка на опт (форма: количество, комментарий) — MVP как lead, без оплаты

### 2.5 Пользователи и авторизация

- Регистрация по email/телефону
- Роли: гость, покупатель, продавец, модератор, администратор
- OAuth (Google) — фаза 2

### 2.6 Админ-панель

- Модерация объявлений и продавцов
- Управление категориями, баннерами, SEO-страницами
- Базовая аналитика (просмотры, регистрации, лиды)

## 4. Бизнес-модель

### 4.1 MVP (бесплатно для привлечения)

- Бесплатная публикация объявлений (лимит: 10 активных на продавца)
- Бесплатный просмотр каталога

### 4.2 Монетизация (после MVP)

| Модель | Описание |
|--------|----------|
| PRO-подписка продавца | Больше объявлений, приоритет в выдаче, бейдж «PRO» |
| Поднятие объявления | Разовая оплата за топ позиции в категории |
| Рекламные баннеры | Главная, категории |
| Верификация «Проверенный поставщик» | Платная проверка документов |
| Лиды | CPL за квалифицированные заявки (фаза 3) |

## 5. Нефункциональные требования

### 5.1 Производительность

- TTFB главной страницы < 500 ms (CDN + SSR)
- Поиск: ответ API < 300 ms при 100k объявлений
- Lighthouse Performance ≥ 85 (mobile)

### 5.2 Масштабируемость

- Горизонтальное масштабирование Next.js через Docker
- PostgreSQL + connection pooling (PgBouncer)
- Redis для кэша поиска и сессий (фаза 2)

### 5.3 Безопасность

- HTTPS everywhere
- Rate limiting на API
- CSRF-защита форм
- Валидация загрузок (только изображения, лимит 5 MB)
- Хранение паролей: bcrypt/argon2
- RBAC по ролям

### 5.4 Доступность

- Семантическая разметка
- Контраст WCAG AA
- Клавиатурная навигация в формах и фильтрах

## 8. Риски и допущения

| Риск | Митигация |
|------|-----------|
| Низкое доверие к новой площадке | Верификация продавцов, отзывы |
| Дубли объявлений | Дедупликация по хешу фото + модерация |
| Спам и фейковые объявления | Модерация, rate limit, жалобы |
| Медленный интернет в регионах | Оптимизация изображений (WebP, lazy load) |

**Допущение:** на этапе MVP сделки совершаются вне платформы (звонок/мессенджер); Tutopt — витрина и генератор лидов.
