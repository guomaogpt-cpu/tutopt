# Prompts

Промпты для Cursor и AI-ассистентов при разработке Tutopt.

## Назначение

- Переиспользуемые инструкции для типовых задач
- Контекстные промпты для feature-разработки
- Шаблоны code review и рефакторинга

## Формат

```markdown
# Prompt: Название

## Контекст
...

## Задача
...

## Ограничения
- Не менять ...
- Следовать docs/...

## Ожидаемый результат
...
```

## Правила

- Ссылаться на `docs/` и `decisions/` для архитектурного контекста
- Не дублировать `docs/00_MASTER_PROMPT.md` — дополнять его
- Именование: `feature-<name>.md`, `review.md`, `refactor-<area>.md`
