# Sprint Overview — Tutopt MVP

## Продукт
Tutopt — B2B classified platform для оптовых объявлений в Кыргызстане.

## Принципы backlog
- Одна задача = один измеримый результат
- Сначала инфраструктура → auth → домены → публичка → admin → SEO → production
- Запрещено: корзина, заказы, оплата, логистика, тендеры, импорт, CRM, ERP
