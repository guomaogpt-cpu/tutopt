# Features

Доменные модули приложения ВсеТут. Каждая папка — изолированная feature-область с собственными компонентами, типами и публичным API модуля.

## Принципы

- Одна feature = один бизнес-домен
- Импорт между features — только через публичный `index.ts` (будет добавлен позже)
- Общий код выносится в `src/shared`
- UI-страницы и route handlers живут в `src/app`, не здесь

## Модули

| Папка        | Назначение                                       |
| ------------ | ------------------------------------------------ |
| `auth`       | Регистрация, вход, восстановление пароля, сессии |
| `users`      | Профиль покупателя, настройки аккаунта           |
| `sellers`    | Профиль компании, кабинет продавца, верификация  |
| `listings`   | CRUD объявлений, галерея, статусы публикации     |
| `categories` | Дерево категорий, навигация по каталогу          |
| `search`     | Поиск, фильтры, сортировка                       |
| `favorites`  | Избранные объявления                             |
| `reports`    | Жалобы на объявления                             |
| `admin`      | Модерация, управление контентом                  |

## Структура feature (шаблон)

```
features/<name>/
├── components/   # UI-компоненты домена
├── hooks/        # React hooks домена
├── types/        # Типы домена
├── utils/        # Утилиты домена
└── index.ts      # Публичный API модуля
```
